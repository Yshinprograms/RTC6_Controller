SCANLAB version information for the RTC6 DLL
--------------------------------------------
SCANLAB Versionsinfo zur RTC6 DLL

***************************************************************************

The DLL file RTC6DLL.dll of version 648 can only be combined with the 
following program files:

 - RTC6OUT.out (DSP program file for RTC6e)     at least version 651
 - RTC6ETH.out (DSP program file for RTC6eth)   at least version 651
 - RTC6RBF.rbf (firmware file)                  at least version 643
 - RTC6DAT.dat (binary support file)            at least version 603


Further instructions for using the RTC6 DLL are given in the RTC6 manual.

-------------------------------------------------------------------------

Die DLL-Datei RTC6DLL.dll der Version 648 kann nur mit den
folgenden Programmdateien kombiniert werden:

 - RTC6OUT.out (DSP-Programmdatei für RTC6e)    mindestens Version 651
 - RTC6ETH.out (DSP-Programmdatei für RTC6eth)  mindestens Version 651
 - RTC6RBF.rbf (Firmwaredatei)                  mindestens Version 643
 - RTC6DAT.dat (binäre Hilfsdatei)              mindestens Version 603


Weitere Informationen zur RTC6 DLL finden Sie im RTC6-Handbuch.
